# Airport Departure Board

Airport Departure Board は、羽田空港第3ターミナルおよび成田空港第2ターミナルから都心方面へ向かう鉄道・バスを、発車時刻順に一覧表示するためのWebアプリです。

複数の事業者・路線の時刻表を1画面にまとめ、空港から出発するときに「次にどれに乗れるか」を確認しやすくすることを目的としています。

## 公開URL

https://hndpvg.github.io/departure-board/

## 主な機能

- 羽田空港 第3ターミナルの発車案内
- 成田空港 第2ターミナルの発車案内
- 京急、東京モノレール、京成、JR、エアポートバス東京・成田に対応
- 平日・土休日の切り替え
- 表示する駅・種別・行先のフィルタ設定
- 参考到着地点の時刻表示
- PWA対応
- オフライン時は最後に取得したデータを表示

## データ出典

各社公式時刻表を基に生成した静的データを使用しています。

- 京浜急行電鉄
- 東京モノレール
- 京成電鉄
- JR東日本
- エアポートバス東京・成田

アプリ実行時は外部APIへアクセスせず、リポジトリ内のJSONデータを読み込みます。

## 制約

- リアルタイム運行情報は反映しません。
- 遅延、運休、番線変更は反映しません。
- 臨時列車や臨時ダイヤは反映されない場合があります。
- 祝日、年末年始、特別ダイヤの自動判定は未対応です。
- 参考到着地点の時刻は、取得できた発車・到着時刻のみ表示します。
- 最新情報は各事業者の公式案内を確認してください。

## 免責事項

本アプリは非公式です。

表示内容の正確性には努めていますが、実際の運行情報や時刻は各事業者の公式情報を確認してください。

本アプリの利用によって生じた損害について、作者は責任を負いません。

## ローカルでの起動

Windowsでは、リポジトリ直下の `start.bat` をダブルクリックすると、ローカルWebサーバーを起動して既定のブラウザを開きます。

手動で起動する場合は、リポジトリのルートで次のコマンドを実行してください。

```powershell
py -m http.server 4173
```

その後、ブラウザで `http://localhost:4173` を開きます。

`py` が利用できない環境では、次のコマンドでも起動できます。

```powershell
python -m http.server 4173
```

## データ更新

公式時刻表を参照してデータを更新する場合は、次のコマンドを実行します。

```powershell
node scripts/update-real-data.mjs
```

更新後は、JSON構造とデータ方針を検証します。

```powershell
node scripts/validate-data.mjs
```

データ管理の詳細は [docs/data-management.md](./docs/data-management.md) を参照してください。
